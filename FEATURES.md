# NesoBot Özellikleri

## 🎯 Ana Özellikler

### 1. Butonlu Çekiliş Sistemi
- Çekiliş oluşturma ve yönetme
- Butonlu katılım sistemi
- Kazanan seçimi ve duyurusu
- Çekiliş geçmişi

### 2. Butonlu Ticket Sistemi
- Kullanıcıların ticket açabilmesi
- Ticket kategorisi ayarı
- Ticket kapatma ve arşivleme
- Ticket mesaj geçmişi

### 3. Komut Sistemi
- `!yardım` - Tüm komutları göster
- `!çekiliş` - Çekiliş başlat
- `!ticket` - Ticket aç
- Kustomize edilebilir ön ek

### 4. Permission Sistemi
- Administrator
- Moderator
- Member
- Özel izinler

### 5. Oto Cevap Sistemi
- Tetikleyiciler ayarlama
- Cevap mesajları
- Regex desteği

### 6. Oto Rol Sistemi
- Sunucuya katılan üyeler için otomatik rol
- Rol seçimi

### 7. Link & Spam Engelleme
- Link engelleme
- Anti-spam
- Ban word sistemi

### 8. Professional Dashboard
- Modern ve responsive tasarım
- Dark mode
- Tailwind CSS

### 9. Discord OAuth2 Entegrasyonu
- Güvenli giriş
- Sunucu yönetimi
- Kullanıcı bilgileri

### 10. Multi-Sunucu Desteği
- Birden fazla sunucu yönetimi
- Sunucu başına ayarlar

## 🛠️ Teknoloji Stack

### Backend
- Node.js 16+
- Express.js
- MongoDB
- Discord.js
- JWT Authentication

### Frontend
- React 18
- React Router
- Tailwind CSS
- Axios

### Bot
- Discord.js 14
- MongoDB
- Slash Commands
