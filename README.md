# NesoBot - Professional Discord Bot Dashboard

Profesyonel Discord bot yönetim sistemi. Butonlu çekiliş, ticket sistemi, permission sistemi ve daha fazlası.

## 🎯 Özellikler

- ✅ Butonlu Çekiliş Sistemi
- ✅ Butonlu Ticket Sistemi
- ✅ !yardım Komutu
- ✅ Permission Sistemi
- ✅ Oto Cevap Sistemi
- ✅ Oto Rol Sistemi
- ✅ Link/Spam Engelleme
- ✅ Professional Dashboard
- ✅ Discord OAuth2 Giriş
- ✅ Multi-Sunucu Yönetimi

## 📁 Proje Yapısı

```
NesoBot/
├── backend/              # Node.js + Express API
├── frontend/             # React Dashboard
├── bot/                  # Discord.js Bot
├── database/             # MongoDB Şeması
├── docker-compose.yml
└── README.md
```
